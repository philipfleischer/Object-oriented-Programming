\doxysubsection{Detailed Description}
A doubly linked list class, where each node holds an \doxylink{class_array_list}{Array\+List} object. 

This class behaves similarly to \doxylink{class_linked_list}{Linked\+List}, but each element in the list is an instance of the datastructure \doxylink{class_array_list}{Array\+List}.
