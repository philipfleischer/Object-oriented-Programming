\doxysubsection{Detailed Description}
A dynamic array-\/based list that grows automatically as elements are added. 

The \doxylink{class_array_list}{Array\+List} stores integers and uses a fixed-\/size underlying array. When capacity is reached, the internal array is resized using a growth factor. It supports indexing, element retrieval, and printing.
