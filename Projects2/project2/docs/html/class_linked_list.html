Detailed Description

A doubly linked list object of class LinkedList.

It supports push to front, append to back, printing, and length queries. The list maintains pointers to both head and tail to make front/back operations efficient.

Inserts a value at a specific index in the list.

The element at the given index and all elements to the to its right will be shifted one position forward. Insertion at head is the same as push_front() and insertion at tail is the same as append().
